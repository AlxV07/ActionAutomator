# ActionAutomator

Flexible Java GUI Application for fast &amp; easy global keyboard + mouse automation.

GUI help in-application.

### Action Code Interface Documentation

Commands:
```
lclick()
lup()
ldown()
rclick()
rup()
rdown()
kpress(key)
kup(key)
kdown(key)
type(text)
wait(time)
setspeed(time)
repeat(nof_times) {}
run(action)
move(x,y)
savestr(name str)
saveint(name,int)
```
